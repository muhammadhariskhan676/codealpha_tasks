# Age Calculator

A simple and responsive Age Calculator built with HTML, CSS, and JavaScript. This application calculates a user's exact age in years, months, and days based on their date of birth.

## Features

- Calculate exact age
- Displays years, months, and days
- Simple and clean user interface
- Responsive design for desktop and mobile
- Instant calculation
- Input validation

## Technologies Used

- HTML5
- CSS3
- JavaScript

## How to Run

1. Clone the repository.

```bash
git clone https://github.com/muhammadhariskhan676/codealpha_tasks.git
```

2. Open the project folder.

```bash
cd codealpha_tasks/Age\ Calculator
```

3. Open `index.html` in your browser.

No installation or dependencies are required.
